# Formula

A simple 3D wireframe cube visualization using HTML5 Canvas and vanilla JavaScript.

Based on [tsoding/formula](https://github.com/tsoding/formula).

## Demo

Opens a canvas displaying a rotating 3D wireframe cube with a retro green-on-black terminal aesthetic.

## Usage

Open `index.html` in any modern web browser.

## Features

- 3D to 2D perspective projection
- Rotation animation around the XZ axis
- Wireframe rendering
- No dependencies - pure vanilla JavaScript
